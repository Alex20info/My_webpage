my projects

Mobile aplication of car representative type

This mobile application is client-server, where the client is the user of the mobile device, and the server has the functionality to validate the client's data and save them in a database, as well as to log the client with personal data. The client part was implemented in java in android studio, and the server part which is divided into two segments, namely: the user login part was implemented in the PHP language on the PHP STorm development environment, for viewing the data I used by XAMPP to launch the user database in PHPMyAdmin. The data validation part was created in Phyton, in the visual studio code development environment.
